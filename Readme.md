Login URL : /api/v1/myauth/login/

Sample Login Parameters:
    {
            "username":"john",
            "password":"johnpassword"
    }

Registration URL: /api/v1/myauth/register/

Sample Registration Parameters:
    {
        "username": "abir",
        "password": "adfa",
        "email": "abir@a.com",
        "first_name": "Abir",
        "last_name": "Ahmed"
    }

Logout URL: /api/v1/myauth/logout/

Sample LogOut Parameters:
    {
            'token': self.token
    }
